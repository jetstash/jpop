JPop

Usage

$('body').jpop(options);

Demo

Positioning

Popover

Displayed as a modal and centered horizontally and vertically on the page.

Banner

Fixed positioning on either the top or the bottom of the page.

Options

Option Default Description
form null Set your Jetstash form ID.
type "banner" Set which type of positioning you want for the popover.
position "top" Set where on the page you want the banner to appear (only for banner type).
style false This is a description.
zindex 8675309 Set the z-index of the popover/jpop-banner.
show false This is a description.
delay 500 Set your delay for the timed option instead of displaying on scroll.
scroll false Set to "true" to enable the popover to appear on scroll; will disable the delay time interval option.
title "Subscribe" Set the popover title text.
button "Submit" Set the popover submit button text.
thanks "Thanks" Thank you message displayed after a user signs up.
cookie true Optionally disable cookies, this prevents users from getting a popup on every page.
expires 2592000 Set the expiration date in seconds of the cookie, defaults to 30 days.

Events

$(window).on("event-name", function(evt) {
  console.log(evt);
});
Name Description
jpop-displayed Fires when the modal/banner is displayed.
jpop-dismissed Fires when the modal/banner is closed.
jpop-success Fires when the ajax submission was successful.
